### What is ced · watch?

I build small, autonomous, offline-first tools that respect your privacy.  
No tracking. No accounts. No ads. No bullshit.

Every app is a clean, self-contained node you fully control.

- **CigTracker** & **SmokeTracker** – Simple cigarette / smoking trackers
- **MindOrbit** – 3D mind mapping tool
- **Pwd-Audit** – Local password security audit
- **NetWatch** – Personal MySpeed dashboard for Umbrel
- **BlockBoard.live** – Clean Bitcoin dashboard (Blockclock style)
- **Juke It** – Minimalist local media player

All projects are **Free & Open Source** · **Donationware**
